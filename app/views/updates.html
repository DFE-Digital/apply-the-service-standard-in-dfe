Styling and rebuild

On 17 May 2023 by Andy Jones, Head of Design

We have published a rebuilt version of the service which uses our new DfE Frontend and header style.

The previous version was built using the GOV tech docs manual template and published to GitHub pages. We had identified a problem with using GitHub pages and as a result have rebuilt the codebaseand moved away from this solution.

Being able to do this, means it's easier to contribute to and manage change publishing through a new pipeline process.

For information, we haven't updated any of the guidance.
